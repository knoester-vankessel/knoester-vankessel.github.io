# Emballage

In de Backoffice kunt u zelf emballages toevoegen. Dit doet u als volgt:

import { Steps } from "nextra/components";
import { Callout } from "nextra/components";

<Steps>

### Open instellingen

Ga in de backoffice naar de rechterbovenhoek en klik op de knop **instellingen**.

### Ga naar Emballage

Open in het tablad **Artikelen** het venster **Emballage**.

### Klik op de knop "âž• Nieuwe emballage"

Hiermee opent u de pop-up **Emballage bewerken**.

</Steps>

## Gegevens invullen

# Uitleg Emballagevelden

| **Veld**                         | **Beschrijving**                                                     |
| -------------------------------- | -------------------------------------------------------------------- |
| **Status**                       | Actief/inactief. Bepaalt of de emballage gebruikt kan worden.        |
| **Code**                         | Unieke interne code van de emballage (bv. H1 QS).                    |
| **EAN code**                     | Barcode/EAN-nummer, optioneel voor scannen of koppeling.             |
| **Omschrijving**                 | Volledige naam/beschrijving van de emballage.                        |
| **Volgorde**                     | Cijfer dat de volgorde bepaalt in lijsten of rapportages.            |
| **Bedrag**                       | Waarde of borgbedrag van de emballage (bv. 55,00).                   |
| **Externe code**                 | Optionele code voor koppeling met externe systemen.                  |
| **Verberg op pakbon**            | Emballage niet tonen op de pakbon als dit is aangevinkt.             |
| **Verberg op retour afleverbon** | Emballage niet tonen op retourdocumenten als dit is aangevinkt.      |
| **Verberg in de Transport APP**  | Emballage niet zichtbaar in de transportapp voor chauffeurs.         |
| **Negatief saldo niet toestaan** | Voorkomt dat er een negatief saldo voor deze emballage kan ontstaan. |
