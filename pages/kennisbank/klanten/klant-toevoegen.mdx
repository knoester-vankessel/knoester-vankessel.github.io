# Klant toevoegen

In dit scherm kun je een nieuwe klant toevoegen aan het systeem. Het formulier is opgedeeld in verschillende secties:

- Klantgegevens
- Factuuradres
- Registratie

![ecbase](/images/klanten/klant_toevoegen.gif)

---

## 🔹 Klantgegevens

| Veld                   | Beschrijving                                                            |
| ---------------------- | ----------------------------------------------------------------------- |
| **Klantnummer**        | Automatisch gegenereerd nummer of handmatig ingevoerd, uniek per klant. |
| **Klanttype**          | Selecteer of het om een zakelijke of particuliere klant gaat.           |
| **Klantnaam**          | De officiële naam van de klant of het bedrijf.                          |
| **Naam 2**             | Eventueel tweede naam of aanvulling (zoals afdeling of vestiging).      |
| **Bankrekeningnummer** | Het bankrekeningnummer (indien geen IBAN).                              |
| **IBAN nummer**        | Internationaal bankrekeningnummer voor SEPA-transacties.                |
| **BIC code**           | Bank Identifier Code, nodig voor internationale betalingen.             |
| **KVK nummer**         | Kamer van Koophandel-nummer van het bedrijf.                            |
| **BTW nummer**         | BTW-identificatienummer van de klant.                                   |
| **Website**            | URL van de website van de klant (optioneel).                            |
| **Vertegenwoordiger**  | Interne medewerker die verantwoordelijk is voor deze klant.             |
| **Telefoonnummer**     | Vast of mobiel telefoonnummer van de klant.                             |
| **Faxnummer**          | Eventueel faxnummer.                                                    |
| **Mobiel nummer**      | Mobiel nummer indien apart van vast nummer.                             |

> ℹ️ Velden gemarkeerd met een rood pijltje zijn **verplicht**.

---

## 🧾 Factuuradres

| Veld                     | Beschrijving                                         |
| ------------------------ | ---------------------------------------------------- |
| **Ter attentie van**     | Naam van de persoon of afdeling binnen het bedrijf.  |
| **Straatnaam**           | Straatnaam van het factuuradres.                     |
| **Huisnr. / Toevoeging** | Huisnummer en eventuele toevoeging (bijv. A of bis). |
| **Postcode**             | Postcode van het factuuradres.                       |
| **Plaats**               | Stad of dorp van het adres.                          |
| **Provincie**            | Selecteer een provincie binnen Nederland.            |
| **Land**                 | Selecteer het land. Standaard is dit Nederland.      |

---

## 📅 Registratie

| Veld                  | Beschrijving                                                          |
| --------------------- | --------------------------------------------------------------------- |
| **Datum registratie** | De datum waarop deze klant is toegevoegd. Wordt automatisch ingevuld. |

---

## ✅ Acties

- **Opslaan** – Slaat de klantgegevens op.
- **Annuleren** – Annuleert de actie en sluit het venster zonder op te slaan.

---

## 📌 Opmerkingen

- Zorg ervoor dat verplichte velden zijn ingevuld om fouten te voorkomen.
- Controleer IBAN/BIC bij internationale klanten.
- Klantnummer is meestal automatisch, tenzij anders ingesteld.
