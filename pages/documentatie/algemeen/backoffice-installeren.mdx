# Backoffice installeren

import { Steps } from 'nextra/components'
 
<Steps>

### Browser installeren

De backoffice van ecBase werkt het meest stabiel in de Chrome browser van Google. Deze kunt u [hier downloaden.](https://www.google.com/intl/nl_nl/chrome/)

### URL openen

Daarna opent u Chrome en gaat u naar de volgende URL: https://app.ecbase.nl/backoffice/

### Snelkoppeling toevoegen

Ga naar de '3 verticale bolletjes' om de opties uit te vouwen. Klik op 'Meer hulpprogramma's' en hierna op 'Snelkoppeling maken...'

Geef een naam (bijvoorbeeld: ecBase Backoffice) en klik op 'Maken'.

</Steps>
